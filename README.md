# OIIO-PythonBindings

Python bindings for OpenImageIO. It uses [Conan 1.59.0](https://conan.io) to build the custom recipe
you can find in *openimageio/conanfile.py*.

## Windows 

### Building
```bat
mkdir oiiobindings
cd oiiobindings
./build.bat
```

### Usage
```bat
./setup_env.bat
python
>>> import OpenImageIO as oiio
>>> ...
```
