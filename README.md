# OIIO-PythonBindings

Python bindings for OpenImageIO with python 3.10. It uses (Conan)[https://conan.io] to build the custom recipe
you can find in *openimageio/conanfile.py*

## Windows 

### Building
```bat
mkdir oiiobindings
cd oiiobindings
./build.bat
```

### Usage
```bat
./setup_env.bat
python
>>> import OpenImageIO as oiio
>>>...
```
